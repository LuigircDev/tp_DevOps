# API python
## RUN local
1. Activate environment
```
python3 -m venv path/to/venv
source path/to/venv/bin/activate
pip3 install -r requirements.txt #install libs
```
2. Run api
```
python3 api.py
```
3. Exit
```
deactivate
```
